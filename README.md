# Todo App (Java + Spring Boot + MySQL)

A full-stack **TODO application** built with **Spring Boot**, **Spring Data JPA**, **MySQL**, and **REST API**.

## Features
- Create, read, update, delete todos
- Persistent storage with MySQL
- RESTful API endpoints

## Tech Stack
- **Java 17+**
- **Spring Boot 3.x**
- **Spring Data JPA (Hibernate)**
- **MySQL 8.0**
- **Maven**
- **Lombok**
